About Me

Jana Haidar

I am a Computer Science student on a continuous learning journey toward becoming an AI engineer. This website represents my second experience in building web projects. In my previous project, which focused on water pollution, I achieved a top-three ranking, motivating me to further develop my skills and creativity. Working on this project added significant value to my learning path. While developing it, I discovered and applied many HTML, CSS, and structuring techniques that I had not used before. Beyond technical growth, this project reflects my personal preferences in layout organization and color palette selection, allowing me to combine functionality with a design style that truly represents me.

Project Purpose

Educational Mission

To create an accessible digital resource that provides visitors, with detailed information about their history, significance, and cultural impact. Through this website, I aim to inspire a deeper appreciation for architectural marvels and historical landmarks that have shaped our world, and serve a virtual museum for students, history enthusiasts, and travelers alike.

Techincal Demonstration

To showcase modern web development skills including responsive design, CSS Grid/Flexbox layouts, interactive elements, and clean code architecture. This project demonstrates proficiency in front-end development best practices.

Cultural Preservation

To contribute to digital cultural preservation by creating an engaging platform that highlights UNESCO World Heritage sites and promotes awareness about the importance of protecting historical landmarks.

Tools & Technologies Used

Frontend Development

This project was developed using HTML5 for structure, CSS3 for layout and visual design, and JavaScriptfor simple interactive features. External resources such as Google Fonts and Font Awesome were used to enhance typography and icons, ensuring a consistent and elegant user experience.
